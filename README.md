사용자에게 PWA를 쉽게 설치할 수 있도록 합니다.

## Directory Structure

FSD architecture

```bash
src /
      app/ (페이지 관련 코드)
      features/ (기능 관련 코드)
      shared/ (공통 코드)
      views/ (페이지 세부 코드)
```
